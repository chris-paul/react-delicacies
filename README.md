### 项目运行

```

$ git clone

$ cd react-delicacies

$ npm install

/*  项目启动 */
$ npm start

/*  项目打包 */
$ npm run build

/*  项目测试用例 */
$ npm run jest

/*  项目测试覆盖率查看 */
$ npm run coverage

/* 打包分布图 */
$ npm run report

```
