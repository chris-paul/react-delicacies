### 安装扩展editorconfig(cmd+shift+p)

```javascript
/* 这是项目的根目录 */
root = true
/* 所以的文件都使用这个规则 */
[*]
charset = utf-8
/* tab 以空格的形式存在 */
indent_style = space
indent_size = 4
/* 航末的结束标志 */
end_of_line = lf
/* 代码结束的时候,如果文件末尾没有空行自动添加 */
insert_final_newline = true
/* 忽略代码行末的空格 */
trim_trailing_whitespace = true
```
